# Proyecto CMB

Este repositorio contiene el notebook y las cadenas MCMC obtenidas con Cobaya para el análisis cosmológico del modelo LCDM utilizando datos de Planck, supernovas y BAO.

## Archivos
- `NO_CMB.ipynb`: notebook principal
- `chains/`: cadenas MCMC generadas
- archivos `.yaml`: configuración usada para la corrida

## Requisitos
Python + Cobaya + CAMB + Healpy
Tener instalados los datos !cobaya-install cosmo -p /path/to/packages
